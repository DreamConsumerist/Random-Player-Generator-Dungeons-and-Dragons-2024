# Random Player Generator (DnD 2024)

In my Delvers home campaign that I run for my friends, I've added a Colosseum facility to the adventuring city that the party operates out of. This is meant to be a PvP gameplay opportunity where, rather than fight against stat blocked monsters, players could have a fair fight.

My first attempt at using this facility was a mess as I didn't have character sheets readily available and I cobbled something together in the moment. I could have made character sheets ahead of time, but that's incredibly timeconsuming and I have enough NPCs that I've handcrafted. I want the spark of randomness and unpredictability.

This project will randomly generate fully valid character sheets for DnD 2024. My initial target is to make these random goons for my players to fight with condensed actionable abilities, but this could easily extend to fully fleshed out sheets. If the UI is pleasant enough, I could make this a full character creator and sheet manager. We'll see how far I get.
